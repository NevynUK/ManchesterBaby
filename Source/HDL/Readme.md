# Hardware Design Language

This directory holds some third party System Verilog implementations of the Manchester Baby and associated tools.  These tools have been downloaded from [University of Cambridge ECAD and Architecture Practical Classes web site](https://www.cl.cam.ac.uk/teaching/1011/ECAD+Arch/background/baby.html).
